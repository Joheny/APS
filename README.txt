# 高级计划与排程系统
## APS(Advanced Planning and Scheduling)
建模：
* 需求(订单、主计划)
* 物料
* 物料清单
* 指令单
* 工艺路线
* 人
* 设备
* 模具
* 资源(人、设备、模具)
* 分部
* 车间
* 工组
* 资源日历
* 资源类别
* 工艺占用 关联工艺、资源日历、时间
* 工艺资源 一个工艺需要用到哪些资源
